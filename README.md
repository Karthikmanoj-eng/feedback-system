# Feedback System

🚀 My first full-stack web application built with HTML, CSS, JavaScript, Node.js, Express, MongoDB Atlas, and Mongoose.

This application allows users to submit feedback, view reviews, and manage stored feedback through a RESTful API.

## Features

* Submit feedback through a simple interface
* Store feedback in MongoDB Atlas
* View all submitted reviews
* Delete stored feedback
* Responsive and modern design
* Full-stack architecture

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## What I Learned

* REST APIs
* CRUD operations
* Frontend ↔ Backend communication
* MongoDB database integration
* Express routing
* Environment variables with `.env`
* Git and GitHub workflows

## Getting Started

```bash
npm install
node server.js
```

Create a `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

Open:

```
http://localhost:5000
```
